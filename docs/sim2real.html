Running sim2real for arm

What is Sim2Real?

In this section we will train the simple which will learn inverse kinematics for the arm using RL(TD3). The input to the RL agent is state(joint angles of arm) & goal location(x,y,z) and the control action is the change in each joint angles to achieve the desired goal.

For TD3 implementation , we used the publicly available code.

Installation

Activate the virtual environment.

virtualenv_name="pyenv_pyrobot"
VIRTUALENV_FOLDER=~/${virtualenv_name}
source ~/${virtualenv_name}/bin/activate

Install Dependencies

pip install gym
pip install pybullet

Train the agent

cd ~/low_cost_ws/src/LoCoBot/sim2real
python train.py --exp_name expID01 --valid_goals --save_models

Before running on the real robot, make sure robot is connected. To connect to robot, open new terminal and run

roslaunch locobot_control main.launch use_arm:=true

Test the above trained policy on real robot

python test.py --use_real_robot --valid_goals --directory ./log/expID01/pytorch_models --file_name LocoBotEnv-v0_0
